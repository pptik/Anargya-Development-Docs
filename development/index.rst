Wings Framework
"""""""""""""""""

Antarmuka dari masing-masing node utama, khususnya yang menggunakan Antarmuka website berjalan diatas mesin java menggunakan framework Wings (http://wings.loonydev.com/). Akan lebih efektif jika Anda mengenal dan menjelajah proses pengembangan menggunakan Wings.

* Get started : ikuti instruksinya di http://wings.loonydev.com/docs/

* Disarankan untuk menggunakan Intellij IDEA, karena sumber kode yang sudah ada mempunyai struktur `.idea` , jika Anda ingin mencoba starter project Wings Framework dengan Intelij, berikut langkah-langkahnya :
        1. Clone atau download : https://github.com/anjulgarg/wings-sample-app
        2. Import menggunakan Intellij, pilih setting default ketika proses import
        3. Cari folder `lib`, Anda akan menemukan beberapa file dengan extensi `.jar`, klik kanan, kemudian pilih `Add as a Library` dan pilih main project dimana lib ini akan ditambahkan, lakukan hal yang sama untuk file `.jar` lainnya.
        4. Pastikan tidak ada error pada console. *Run* -> *Edit Configuration*, kemudian tambahkan tomcat server untuk menjalankan programnya dan set `war explode`.
